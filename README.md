# Leo

Leo 시리즈 앱 모음입니다. 아래에서 원하는 앱의 APK를 받아 설치하세요.

## DrawLeo

어린이를 위한 안드로이드 태블릿용 그림 그리기 앱입니다. 손끝이나 펜으로 자유롭게 그리고, 붓 크기·색상을 바꿔가며 그림을 저장할 수 있습니다.

- APK 내려받기: <https://proxy.somuna.net/apk-releases/drawleo/drawleo.apk>
- 자세한 소개: [`drawleo/README.md`](https://github.com/previc/drawleo/blob/v2.2.1/README.md)

## ListenLeo

유튜브에서 가져온 오디오를 라이브러리로 정리해 듣는 안드로이드 앱입니다. 가져오기 → 라이브러리 관리 → 재생까지 하나의 흐름으로 이용할 수 있습니다.

- APK 내려받기: <https://proxy.somuna.net/apk-releases/listenleo/listenleo.apk>
- 자세한 소개: [`listenleo/README.md`](https://github.com/previc/ListenLeo/blob/v1.9.10/README.md)

## PlayLeo

어린이를 위한 안드로이드 교육 게임 허브 앱입니다. 포켓몬 도감(1025마리, 한/영 이름, 타입 배지, TTS 발음 재생, 세대 필터) 등 여러 미니 게임을 담습니다.

- APK 내려받기: <https://proxy.somuna.net/apk-releases/playleo/playleo.apk>
- 자세한 소개: [`playleo/README.md`](https://github.com/previc/playleo/blob/v1.2.0/README.md)
